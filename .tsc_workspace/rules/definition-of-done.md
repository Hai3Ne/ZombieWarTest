# Definition of Done

- Kết quả đáp ứng tiêu chí nghiệm thu và không mở rộng ngoài nhu cầu.
- Code tuân thủ coding-standard.md và có dependency rõ ràng.
- Build không có compile error; tests liên quan pass.
- Gameplay không cấp phát lặp lại hoặc Instantiate/Destroy trong steady state.
- Codex và Claude entrypoints vẫn đồng bộ.
